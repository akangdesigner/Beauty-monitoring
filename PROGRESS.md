# 開發進度

## 專案概覽
美妝競品監控台 — 前後端系統，追蹤各電商平台商品價格與庫存狀態。

## 技術棧
- **前端**：React + Vite（`/frontend`），開發伺服器 http://localhost:5173
- **後端**：Node.js + Express + SQLite（`/server`），API 伺服器 http://localhost:3000
